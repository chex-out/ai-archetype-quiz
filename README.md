# AI Worker Archetype Quiz

A self-contained web app that maps marketing professionals to one of three AI worker archetypes — Cyborg, Centaur, or Delegator — based on research from Harvard, MIT, and BCG. Seven questions, a personalised result generated by Claude, deployed to Vercel and accessible via QR code or shared link on any smartphone.

## Deployment

1. Create a new project on Vercel
2. Upload or connect the `ai-quiz` folder
3. Add environment variable: `ANTHROPIC_API_KEY` = your key
4. Deploy — Vercel will detect the `/api` folder and wire the serverless function automatically
5. Copy the deployment URL and generate a QR code at qr-code-generator.com or similar
6. Test on a real smartphone before the session
